# Master-Dataset-Integration-Location-Based-Analysis

**Stack:** R • dplyr • tidyr • lubridate • ggplot2 • maps

Cleaned and standardized a raw GBIF dataset by resolving inconsistent missing values, fixing date formats, restructuring fields, and validating data integrity.

Integrated external reference data and engineered features (season, hemisphere, time-based metrics) to enhance analysis. Delivered a structured, analysis-ready dataset with visual insights on geographic distribution and long-term trends.
